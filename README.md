## Project

Application listing movies and their detailed information.
Deployed project : [Moviee](https://moviee.netlify.com/)

## How to launch the project

You will need an **API_KEY** from [The Movie Database API](https://developers.themoviedb.org/3) to make the project works.
Type **yarn start** to start the magic.

## Used technologies

- React

  - styled-components _styling_
  - react-router-dom _navigation_
  - react-transition-group _react animation_
  - jest-mock-axios _unit test_
  - contexts & wrapped components _optimization_

- API

  - [The Movie Database API](https://developers.themoviedb.org/3)
